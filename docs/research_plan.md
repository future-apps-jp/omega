# SK計算と複素振幅構造の関係を検証する研究計画書

## 0. 問題の核心：なぜ複素数なのか？

本研究の根本的問いは以下である：

> **なぜ量子力学は実数でも四元数でもなく、複素数を用いるのか？**

従来の「計算から量子へ」のアプローチ（Wolfram Physics Project 等）は、
複素振幅を**外部から解釈として与える**に留まっていた。
本研究は、複素数構造が計算の代数的・幾何学的性質から
**導出されるかどうかを検証する**ことを目的とする。

---

## 0.5 用語の定義：「複素数の導出」とは何を意味するか

本研究において「複素数がSK計算から導出される」とは、以下の**いずれか**が成立することを指す：

| 導出の種類 | 定義 | 検証方法 |
|------------|------|----------|
| **公理的導出** | SK計算の分岐確率がSorkinの2次非加法性条件を満たし、複素数体が**数学的に要請される** | I₂ ≠ 0 かつ I₃ = 0 の検証 |
| **代数的導出** | SK書き換え演算子の代数構造が複素数体 ℂ を**部分代数として含む** | J² = -I を満たす演算子の探索 |
| **幾何学的導出** | パス空間のホロノミー群が U(1) を含み、位相が**幾何学的に決定される** | ホロノミー計算 |

**重要**: 本研究は「導出される」ことを前提としない。
「導出されない」という結論も同等に価値のある成果である。

## 1. 研究目的

本研究の目的は、SKコンビネータ計算（抽象度10）の内在的構造と、
量子力学の複素振幅構造（抽象度9）との間に**関係があるかどうかを検証する**ことである。

具体的には：

1. SK計算の書き換え系が持つ代数的構造から複素数体 ℂ が**現れるかどうかを調べる**
2. 計算パス空間の幾何学的構造から位相 exp(iΦ) の形式が**生じるかどうかを検証する**
3. **Born則**（|ψ|² が確率を与える）との関連性を**探索する**
4. 上記を踏まえた数値実験により、干渉現象との対応を**調査する**

## 1.5 研究の哲学：「否定的結果」の価値

### 本研究の根本的立場

本研究は「複素数が導出できるはず」という楽観的予断に基づくものではない。
むしろ、以下の問いに対する**明確な回答**を得ることを目的とする：

> SK計算から量子的複素構造は現れるか、現れないか？

**いずれの結論も等しく価値がある。**

### 否定的結果の学術的意義

| 否定的結果 | 学術的貢献 |
|------------|------------|
| 「SK計算から複素数は現れない」 | 計算基盤と量子構造の関係に関する**制約条件の確立** |
| 「古典確率で十分である」 | SK計算が本質的に古典的であることの**確認** |
| 「位相は計算的に決定されない」 | 量子の位相構造に関する**独立性の確認** |

これらは、将来の研究者が「どこを探すべきでないか」を知るための**地図**となる。
否定的結果は肯定的結果と同等以上に publishable である。

### 3.1 アプローチA：書き換え演算子の代数構造（優先度：中）

#### 仮説
SK計算の書き換え操作を演算子として捉えると、
その代数的閉包が複素数体を含む**可能性がある**。

#### 先行研究・理論的根拠

**根拠1: Clifford代数との類似性**
- Clifford代数 Cl(n) は、生成元 γᵢ が γᵢγⱼ + γⱼγᵢ = 2δᵢⱼ を満たす
- Cl(2) ≅ ℍ（四元数）、Cl(0,2) ⊃ ℂ（複素数を含む）
- **仮説**: SK の書き換え演算子 Ŝ, K̂ が Clifford 的関係を満たす可能性

**根拠2: 量子群との接続**
- Kauffman (1991) は、結び目不変量と量子群の関係を示した
- SKコンビネータはλ計算と同型であり、λ項は結び目図式で表現可能
- **仮説**: この接続から q-変形代数（複素数を含む）が現れる可能性

**根拠3: 可逆計算の代数**
- Bennett (1973) の可逆計算では、全ての操作に逆が存在
- SK計算の可逆拡張では、Ŝ⁻¹, K̂⁻¹ が定義される
- **仮説**: (ŜK̂)² などの組み合わせが -I を与える可能性

#### 検証方法
1. SK式のクラス [E] 上の演算子代数を構成
2. 生成元 Ŝ, K̂ の関係式を網羅的に導出
3. J² = -I を満たす J の存在を検証
4. Clifford代数 Cl(p,q) への埋め込み可能性を検証

#### 否定的結果の場合
「SK演算子代数は複素数を含まない」
→ 純粋な書き換え操作からは量子位相は生じないことの**確認**
→ 位相の起源は代数的ではなく、幾何学的または情報論的である可能性

### 3.2 アプローチB：パス空間の幾何学（優先度：中）

#### 仮説
SK計算パス空間は自然な**接続（connection）**を持ち、
その**ホロノミー（曲率）**から位相が生じる**可能性がある**。

#### 先行研究・理論的根拠

**根拠1: Berry位相の普遍性**
- Berry (1984) は、量子系のパラメータ空間を一周すると幾何学的位相が生じることを示した
- これは**トポロジカルな性質**であり、系の詳細によらない
- **仮説**: SK計算のパス空間にも類似の幾何学的構造が存在する可能性

**根拠2: ホロノミー群と量子**
- Anandan & Aharonov (1990) は、ホロノミー群 U(1) が量子位相の起源であることを示した
- **仮説**: SK計算のパス空間のホロノミー群が U(1) を含む可能性

**根拠3: 圏論的視点**
- Baez (2006) の higher gauge theory では、2-圏の構造から位相が現れる
- SK計算の multiway graph は自然に 2-圏構造を持つ
- **仮説**: この 2-圏構造のホロノミーが位相を生む可能性

#### 数学的定式化
```
パス空間 P = {SK計算の全簡約パス}
隣接関係: p₁ ~ p₂ ⟺ p₁ と p₂ は1つの redex 選択のみ異なる
ループ: γ = p₁ → p₂ → ... → p₁ （同じ終状態に戻る）
ホロノミー: Hol(γ) ∈ G （ある群 G）

検証: G ⊃ U(1) かどうかを調べる
```

#### 検証方法
1. 具体的な SK 式で multiway graph を構築
2. 同じ終状態に至る複数パスのペアを列挙
3. パスペア間の「差分」を定義（例：使用した redex の差）
4. この差分が U(1) 群の構造を持つか検証

#### 否定的結果の場合
「SK計算のパス空間は自明なホロノミーを持つ」
→ 計算パスの幾何学からは量子位相は生じないことの**確認**
→ 位相の起源は純粋に代数的または情報論的である可能性

### 3.3 アプローチC：情報理論的導出（優先度：低→中）

#### 仮説
計算パスの**アルゴリズム的情報量**の差分が位相を定義する**可能性がある**。

#### 先行研究・理論的根拠

**根拠1: Landauer原理**
- Landauer (1961): 1ビットの情報消去には最低 kT ln 2 のエネルギーが必要
- K-combinator は情報消去を行う（Kxy → x で y を捨てる）
- **仮説**: この「消去コスト」が位相として蓄積される可能性

**根拠2: Wheeler's "It from Bit"**
- Wheeler (1990): 物理的実在は情報から構成される
- 情報量の差が物理量（エネルギー、位相）に対応する
- **仮説**: Kolmogorov複雑性の差分 ΔK が作用（位相の源）に対応する可能性

**根拠3: 計算熱力学**
- Zurek (1989): 計算過程のエントロピー生成と熱力学的コストの関係
- 各簡約ステップには「計算論的コスト」が付随
- **仮説**: このコストの積分が位相 Φ を与える可能性

#### 具体的定式化
```
各パス p に対し:
- 複雑性 K(p) = パスを記述する最短プログラムの長さ
- 消去情報量 E(p) = パス中で K-combinator が消去した情報の総量
- 作用 S(p) = α·K(p) + β·E(p) （α, β は定数）
- 位相 Φ(p) = S(p) / ℏ_eff （ℏ_eff は有効プランク定数）

振幅: f(p) = exp(i·Φ(p))
```

#### 検証方法
1. 各パスの K(p) を近似計算（圧縮アルゴリズム使用）
2. E(p)（消去情報量）を正確に計算
3. S(p) と干渉パターンの相関を統計解析
4. Φ ∝ S となる α, β の存在を検証

#### 否定的結果の場合
「情報量と位相に有意な相関がない」
→ 量子位相は情報量から独立した構造であることの**確認**
→ 位相の起源は純粋に代数的または幾何学的である可能性

### 3.4 アプローチD：確率の非加法性からの逆問題（優先度：🔴最高）

#### 仮説
SK計算の分岐において「パス確率」を定義すると、
古典確率（加法的）では矛盾が生じ、複素振幅が要請される**可能性がある**。

#### 先行研究・理論的根拠（**最も強固**）

**根拠1: Sorkin の定理（1994）**

Sorkin は以下を厳密に証明した：

> **定理**: 2重スリット実験で干渉が生じるのは、確率測度が**2次の非加法性**を持ち、
> かつ**3次以上の非加法性がゼロ**であるとき、そしてそのときに限る。

数学的には：
```
I₂(A,B) = P(A∪B) - P(A) - P(B) ≠ 0  （2次干渉あり）
I₃(A,B,C) = P(A∪B∪C) - P(A∪B) - P(B∪C) - P(C∪A) + P(A) + P(B) + P(C) = 0
            （3次干渉なし）
```

**この条件を満たす最小の数体系は複素数である。**

**根拠2: 一般確率論（GPT）での複素数の特異性**

Chiribella et al. (2011) は、以下を示した：
> 連続可逆性 + 局所トモグラフィー + 純粋化 → 量子論（複素Hilbert空間）

特に「2次非加法性かつ3次加法性」は**量子論に固有**の性質である。

**根拠3: SK計算への適用可能性**

SK計算の multiway graph において：
- 各経路 A, B, C を「スリット」と見なせる
- 終状態 s への「到達確率」を定義できる
- Sorkin の I₂, I₃ を計算できる

**もし I₂ ≠ 0 かつ I₃ = 0 が現れれば、
複素振幅は「手で入れる」のではなく「数学的に要請される」。**

#### 検証方法（最も concrete）

```python
# 擬似コード
def verify_sorkin(sk_expression):
    graph = build_multiway_graph(sk_expression)
    final_states = get_final_states(graph)
    
    for s in final_states:
        paths = get_paths_to(graph, s)
        
        # パスの3つ組に対して Sorkin 公式を検証
        for A, B, C in combinations(paths, 3):
            P_A = path_probability(A)
            P_B = path_probability(B)
            P_C = path_probability(C)
            P_AB = combined_probability(A, B)
            P_BC = combined_probability(B, C)
            P_CA = combined_probability(C, A)
            P_ABC = combined_probability(A, B, C)
            
            I2_AB = P_AB - P_A - P_B
            I3_ABC = P_ABC - P_AB - P_BC - P_CA + P_A + P_B + P_C
            
            if I2_AB != 0 and I3_ABC == 0:
                print("量子的非加法性を検出！")
                return True
    
    return False
```

#### I₂ ≠ 0 が確認された場合

1. **複素数の数学的要請**: Sorkin の定理により、複素振幅が数学的に要請される
2. **Born則への道**: 2次非加法性から |ψ|² の形式が導かれる
3. **アプローチA-Cへの指針**: どの構造が I₂ ≠ 0 を生むか探求

#### I₂ = 0 のみの場合（否定的結果）

全ての確率定義で I₂ = 0（古典的）であれば：

→ SK計算は本質的に古典的であることの**確認**
→ これは**明確な結論**であり、十分に publishable
→ 「計算から量子へ」の経路は SK 以外を探すべきという指針

## 4. Born則との関連

### 4.1 問題設定
なぜ確率は |ψ|² で与えられるのか？

### 4.2 アプローチDとの接続

Sorkin の枠組みでは、2次非加法性の条件から Born 則が導かれる：

```
I₂(A,B) = P(A∪B) - P(A) - P(B)
        = |ψ_A + ψ_B|² - |ψ_A|² - |ψ_B|²
        = ψ_A*ψ_B + ψ_B*ψ_A
        = 2 Re(ψ_A*ψ_B)
```

**つまり、I₂ ≠ 0 が確認されれば、Born則は自動的に導かれる。**

### 4.3 計算資源保存則からの独立導出

#### 仮説
SK計算には「保存される量」が存在し、
それが確率測度の規格化条件を与える**可能性がある**。

#### 具体的候補
- **簡約ポテンシャル**: 式の複雑性から導かれる「エネルギー」的量
- 各パスの「重み」w(p) は保存則から w(p) = |f(p)|² の形に制約される可能性

## 6. 評価指標と結論の判定基準

### 6.1 肯定的結論の判定基準

| 項目 | 基準 | 検証方法 |
|------|------|----------|
| 複素数の要請 | A-D のいずれかで ℂ 構造が現れる | 数学的解析 + 数値検証 |
| 位相の決定 | 位相値が恣意的でなく計算的に決定される | 異なる SK 式での一貫性 |
| Born則 | \|ψ\|² が整合的確率解釈である | Sorkin の定理との対応 |
| 干渉 | 非古典的干渉パターンが出現 | 数値シミュレーション |

### 6.2 否定的結論の判定基準

| 結果 | 意味 | 学術的貢献 |
|------|------|------------|
| A-D 全てで複素構造が現れない | SK計算と量子構造に関係がない | **独立性の確認** |
| 全確率定義で I₂ = 0 | SK計算は本質的に古典的 | **SK古典性の確認** |
| 任意の位相規則で同じ結果 | 位相は計算から決まらない | **位相独立性の確認** |

### 6.3 部分的成功の判定

| 結果 | 意味 | 次のステップ |
|------|------|--------------|
| 一部の確率定義で I₂ ≠ 0 | 特定条件下で量子性出現 | 条件の一般化を探求 |
| J² = -I は発見、ただし位相なし | 代数的構造はあるが動的でない | 時間発展の導入 |
| ホロノミー ≠ U(1) だが非自明 | 位相は存在するが連続でない | 離散位相の研究 |

## 9. 期待される学術的貢献

### 9.1 肯定的結果の場合
- 「なぜ量子力学は複素数を使うか」への計算論的視点

- Digital Physics の未解決問題への新たな知見
- 高インパクトジャーナル（Physical Review Letters, Nature Physics等）への投稿可能性

- 専門誌（Journal of Mathematical Physics, Foundations of Physics等）への投稿可能性

- **「SK計算と量子構造は独立である」という明確な結論**

- タイトル例：*"On the independence of quantum structure from SK combinatory logic"*
