# SK計算による二重スリット干渉ミニモデル：研究計画書

## 1. 研究目的
本研究の目的は、SKコンビネータ（抽象度10）の計算パス分岐から、
量子力学における複素振幅・干渉パターン（抽象度9）が自然に創発するかを検証することである。

具体的には、以下を目指す：

- SK計算の **非決定的分岐（redex選択）** を量子的経路に対応させる  
- 各計算パスに **複素振幅** を割り当てる規則を定義する  
- 終状態における複素振幅の総和から、  
  **干渉（強め合い / 打ち消し）** が生じるかを数値的に検証する  

これは Wolfram Physics Project が到達できなかった  
「計算 → 量子（複素振幅・Hilbert空間）」への橋渡しであり、  
世界的にも未踏の研究領域である。

---

## 2. 背景と意義

### 2.1 SKコンビネータ（抽象度10）
- 最小普遍計算モデル（S, K の2記号のみ）
- λ計算と同等の計算能力
- 書き換えの順序によって **複数の計算パス（分岐）** が発生する  
  → 二重スリットの多経路性と対応

### 2.2 量子力学における複素振幅（抽象度9）
- 電子の二重スリット実験は  
  **複素振幅の重ね合わせ** により干渉パターンが生成される
- 各経路に振幅 f(p) を割り当て、  
  終状態の振幅を ψ(s) = Σ_p f(p) とする
- 古典確率は再現不可能

### 2.3 研究の新規性
- SK計算パスに **自然な複素位相** を割り当てる試みは世界初  
- 経路の干渉を観測できれば、  
  「量子は計算から創発する」という強い証拠となる  
- 抽象度10→9 の橋渡しを初めて構成する意義がある

---

## 3. 研究方法

### 3.1 実験用 SK式（Toy Model）
簡単な分岐構造を持つSK式を採用：

E = S (K a) (K b)


この式は redex の選択順序により  
最終的に **異なる終状態（a または ab）** を生じる。

### 3.2 パスの構成（経路A / 経路B）
- **経路A**：Redex A を先に簡約  
- **経路B**：Redex B を先に簡約

SK計算には自然な「多経路性」が存在するため、  
二重スリットのスリットA/Bに対応付けが可能。

---

## 4. 複素振幅の付与規則 f(p)

### 4.1 基本方針
量子の位相（波のタイミング）を再現するため、  
各パスに複素数の振幅を割り当てる。

### 4.2 振幅の定義
候補として以下を採用：

f(p) = exp(i × Φ(p))


位相 Φ(p) は以下の要因から構成：

- 簡約ステップ数  
- 使用した redex（S or K）  
- パスの構造（深さ・分岐回数）  

**例（ミニモデル用簡易ルール）**
- S-redex： +π/2  
- K-redex： -π/2  
- application： 0  

---

## 5. 例：経路ごとの複素振幅計算

### 経路A：位相 Φ(A) = −π/2 → f(A) = -i  
### 経路B：位相 Φ(B) = +π/2 → f(B) = +i

## 6. 期待される干渉構造

### 6.1 終状態 s の振幅
ψ(s) = Σ_{p→s} f(p)

### 6.2 干渉パターン
複数の経路が同じ終状態へ到達する場合：

- 位相差 0 → 強め合い（明帯）
- 位相差 π → 打ち消し（暗帯）
- 中間 → 干渉縞

### 6.3 目的
より複雑なSK式を用いて  
**同じ終状態に複数パスが収束する状況を作り、干渉の有無を確認する。**
