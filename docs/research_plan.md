# SK計算と複素振幅構造の関係を検証する研究計画書

## 0. 問題の核心：なぜ複素数なのか？

本研究の根本的問いは以下である：

> **なぜ量子力学は実数でも四元数でもなく、複素数を用いるのか？**

従来の「計算から量子へ」のアプローチ（Wolfram Physics Project 等）は、
複素振幅を**外部から解釈として与える**に留まっていた。
本研究は、複素数構造が計算の代数的・幾何学的性質から
**導出されるかどうかを検証する**ことを目的とする。

---

## 0.5 用語の定義：「複素数の導出」とは何を意味するか

本研究において「複素数がSK計算から導出される」とは、以下の**いずれか**が成立することを指す：

| 導出の種類 | 定義 | 検証方法 |
|------------|------|----------|
| **公理的導出** | SK計算の分岐確率がSorkinの2次非加法性条件を満たし、複素数体が**数学的に要請される** | I₂ ≠ 0 かつ I₃ = 0 の検証 |
| **代数的導出** | SK書き換え演算子の代数構造が複素数体 ℂ を**部分代数として含む** | J² = -I を満たす演算子の探索 |
| **幾何学的導出** | パス空間のホロノミー群が U(1) を含み、位相が**幾何学的に決定される** | ホロノミー計算 |

**重要**: 本研究は「導出される」ことを前提としない。
「導出されない」という結論も同等に価値のある成果である。

### 結果の解釈

#### 🔍 発見事項

1. **2パスのケース**: 全て I₂ = 0（古典的加法性）
   - `(K a b) (K c d)`: 2パスが同じ終端に到達、I₂ = 0
   - `S (K a) (K b) c`: 2パスが同じ終端に到達、I₂ = 0

2. **多パスのケース**: I₂ ≠ 0 が観測された
   - `(K a b) (K c d) (K e f)`: 6パス、全15ペアで I₂ ≠ 0
   - ただし I₃ = 0 の3つ組は 0/20

#### ⚠️ 重要な注意

観測された I₂ ≠ 0 は**真の量子的非加法性ではない**。

理由：
- P(A∪B) を「終端への総確率」と定義している
- 6パスが同じ終端に到達する場合、P(終端) = 1
- 各パスの確率 P(A) = P(B) = 1/6
- したがって I₂ = P(A∪B) - P(A) - P(B) = 1 - 1/6 - 1/6 = 2/3 ≠ 0

これは**確率の定義に起因する見かけの非加法性**であり、
量子力学における干渉（振幅の打ち消し合い）とは異なる。

#### 📋 結論

> **現在の確率定義では、SK計算のパス分岐は古典的（加法的）である。**
> 
> 真の量子的非加法性を検出するためには：
> 1. パスに複素振幅を割り当てる方法を導出する必要がある
> 2. アプローチ A-C を探求して、複素構造の起源を調べる

## 1. 研究目的

本研究の目的は、SKコンビネータ計算（抽象度10）の内在的構造と、
量子力学の複素振幅構造（抽象度9）との間に**関係があるかどうかを検証する**ことである。

具体的には：

1. SK計算の書き換え系が持つ代数的構造から複素数体 ℂ が**現れるかどうかを調べる**
2. 計算パス空間の幾何学的構造から位相 exp(iΦ) の形式が**生じるかどうかを検証する**
3. **Born則**（|ψ|² が確率を与える）との関連性を**探索する**
4. 上記を踏まえた数値実験により、干渉現象との対応を**調査する**

## 1.5 研究の哲学：「否定的結果」の価値

### 本研究の根本的立場

本研究は「複素数が導出できるはず」という楽観的予断に基づくものではない。
むしろ、以下の問いに対する**明確な回答**を得ることを目的とする：

> SK計算から量子的複素構造は現れるか、現れないか？

**いずれの結論も等しく価値がある。**

### 否定的結果の学術的意義

| 否定的結果 | 学術的貢献 |
|------------|------------|
| 「SK計算から複素数は現れない」 | 計算基盤と量子構造の関係に関する**制約条件の確立** |
| 「古典確率で十分である」 | SK計算が本質的に古典的であることの**確認** |
| 「位相は計算的に決定されない」 | 量子の位相構造に関する**独立性の確認** |

これらは、将来の研究者が「どこを探すべきでないか」を知るための**地図**となる。
否定的結果は肯定的結果と同等以上に publishable である。

### 3.2 アプローチA：書き換え演算子の代数構造（✅ Phase 1A 完了）

#### 仮説
SK計算の書き換え操作を演算子として捉えると、
その代数的閉包が複素数体を含む**可能性がある**。

#### 先行研究・理論的根拠

**根拠1: Clifford代数との類似性**
- Clifford代数 Cl(n) は、生成元 γᵢ が γᵢγⱼ + γⱼγᵢ = 2δᵢⱼ を満たす
- Cl(2) ≅ ℍ（四元数）、Cl(0,2) ⊃ ℂ（複素数を含む）
- **仮説**: SK の書き換え演算子 Ŝ, K̂ が Clifford 的関係を満たす可能性

**根拠2: 量子群との接続**
- Kauffman (1991) は、結び目不変量と量子群の関係を示した
- SKコンビネータはλ計算と同型であり、λ項は結び目図式で表現可能
- **仮説**: この接続から q-変形代数（複素数を含む）が現れる可能性

**根拠3: 可逆計算の代数**
- Bennett (1973) の可逆計算では、全ての操作に逆が存在
- SK計算の可逆拡張では、Ŝ⁻¹, K̂⁻¹ が定義される
- **仮説**: (ŜK̂)² などの組み合わせが -I を与える可能性

#### 検証方法
1. SK式のクラス [E] 上の演算子代数を構成
2. 生成元 Ŝ, K̂ の関係式を網羅的に導出
3. J² = -I を満たす J の存在を検証
4. Clifford代数 Cl(p,q) への埋め込み可能性を検証

#### 否定的結果の場合
「SK演算子代数は複素数を含まない」
→ 純粋な書き換え操作からは量子位相は生じないことの**確認**
→ 位相の起源は代数的ではなく、幾何学的または情報論的である可能性

## 4. Born則との関連

### 4.1 問題設定
なぜ確率は |ψ|² で与えられるのか？

### 4.2 Phase 0 からの知見

Sorkin の枠組みでは、2次非加法性の条件から Born 則が導かれる：

```
I₂(A,B) = P(A∪B) - P(A) - P(B)
        = |ψ_A + ψ_B|² - |ψ_A|² - |ψ_B|²
        = ψ_A*ψ_B + ψ_B*ψ_A
        = 2 Re(ψ_A*ψ_B)
```

**Phase 0 の結果**: 複素振幅 ψ なしでは I₂ = 0 となり、Born則は適用不可。
**次のステップ**: アプローチA-C で複素振幅の起源を探り、その後 Born則との対応を検証。

これは**複素係数を最初から導入**しているだけで、**創発ではない**。

**結論**:
> 有限基底上の行列表現では、SK演算子から非自明な複素構造は見つからなかった。
> 基底のサイズや構成方法を変えても、根本的な問題は残る。

## 📋 全体総括：Phase 0-2 の結果

### 各フェーズの結果一覧

| Phase | アプローチ | 目標 | 結果 | 位相の由来 |
|-------|------------|------|------|------------|
| **Phase 0** | Sorkin公式 | I₂ ≠ 0 の検証 | 古典的（I₂=0） | N/A |
| **Phase 1A** | 代数的 | J² = -I の探索 | 自明解のみ | 複素係数を仮定 |
| **Phase 1B** | 幾何学的 | U(1) ホロノミー | 接続依存 | 接続で仮定 |
| **Phase 2** | 情報理論的 | 情報量→位相 | 部分的成功 | 演算数から計算 |

### 最終結論

> **SK計算から複素数構造を「完全に導出」することはできなかった。**

#### 何が確認されたか

1. **SK計算のパス空間は十分に豊か**
   - 複数パス、ループ、非自明なグラフ構造が存在
   - 量子的干渉を記述するための「場」は存在する

2. **位相を「計算」することは可能**
   - Phase 2 で情報量から位相差を計算
   - `linear` 式で非自明な位相差を発見

3. **しかし、全てのアプローチに「任意性」が残る**
   - Phase 1A: 複素係数を最初から仮定
   - Phase 1B: 接続の選択が恣意的
   - Phase 2: 計算式の選択が恣意的

#### 何が確認されなかったか

1. **複素数の「必然性」**
   - SK計算から複素数が「必ず現れる」わけではない

2. **位相の「一意性」**
   - どの方法で位相を定義すべきかが決まらない

3. **物理との対応**
   - 計算された位相が量子力学の干渉と対応するかは未検証

### 学術的貢献（否定的結果として）

この研究は以下の点で学術的に価値がある：

| 貢献 | 説明 |
|------|------|
| **制約条件の確立** | SK計算から量子構造は自動的には現れない |
| **追加仮定の明確化** | 位相を得るには接続/計算式の選択が必要 |
| **方法論の提示** | 代数的・幾何学的・情報理論的アプローチの比較 |
| **将来の道標** | どの方向を探すべきか/探すべきでないかの指針 |

### 推奨される将来の研究方向

1. **接続/計算式の「自然な」選択基準の探索**
   - Landauer原理、エントロピー生成からの制約
   - 物理的な原理から接続を決定できるか

2. **異なる計算モデルでの検証**
   - Lambda計算、チューリングマシン
   - 異なる計算基盤で同じ問題が発生するか

3. **量子計算との接続**
   - 量子コンピュータのゲート操作との対応
   - 測定過程との関連
