# 研究計画 v2: 可逆計算と量子構造の関係（レビュー反映版）

## 0. 前研究のサマリー

### 前研究（v1）の成果

研究計画 v1（2024年12月）では、SK計算と量子構造の関係を4つのアプローチで検証した：

| Phase | アプローチ | 結果 | 結論 |
|-------|------------|------|------|
| Phase 0 | Sorkin公式 | I₂ = 0（2パス）| 古典的 |
| Phase 1A | 代数的構造 | 自明解のみ | 複素係数を仮定しない限り現れない |
| Phase 1B | 幾何学的構造 | 接続依存 | 恣意的な仮定が必要 |
| Phase 2 | 情報理論的 | 部分的成功 | 計算式の選択が任意 |

**最終結論**: SK計算から複素数構造は**自動的には現れない**。
→ 論文 "On the Independence of Quantum Structure from SK Combinatory Logic" として発表準備完了

### 残された問題

1. **接続/計算式の「自然な」選択基準が不明**
   - Phase 1B, Phase 2 で位相は計算可能だが、選択が恣意的
   
2. **可逆性の未検証**
   - SK計算は非可逆（Kが情報を破棄）
   - 量子力学はユニタリ（可逆）
   
3. **他の計算モデルでの検証が未実施**
   - 可逆セルオートマトン、可逆論理ゲート等

## 1. 新研究の動機と理論的位置づけ

### 1.1 なぜ可逆計算か？

前研究の否定的結果から、重要な洞察が得られた：

> **SK計算の非可逆性（K combinator の情報破棄）が、量子構造の非創発の原因かもしれない**

量子力学の特徴：
- ユニタリ時間発展（可逆）
- 測定時のみ非可逆（波束の収縮）

SK計算の特徴：
- K combinator が情報を破棄（非可逆）
- Landauer原理により熱放出が必要
- 計算過程で「痕跡」が失われる

### 1.2 ⚠️ 重要な理論的注意：「可逆性 ≠ 量子性」

**本研究の批判的スタンス**:

> **可逆計算は、量子力学の「半分」（ユニタリ性）しか持っておらず、
> 残り半分の「複素構造（干渉）」を導くには別の原理が必要ではないか？**

この注意が必要な理由：

| 系 | 可逆性 | 構造 | 干渉 |
|----|--------|------|------|
| **古典ハミルトン力学** | ✓ 可逆 | シンプレクティック | ✗ なし |
| **Fredkin/Toffoliゲート** | ✓ 可逆 | 実数・離散 | ✗ なし |
| **量子力学** | ✓ 可逆（ユニタリ） | 複素・ケーラー | ✓ あり |

**古典力学も可逆である**という事実は見落とされがちである。
可逆な系が導くのは通常**シンプレクティック構造**であり、
必ずしも**複素構造**ではない。

したがって本研究は、「可逆なら量子になる」という楽観的仮説ではなく、
**「可逆計算において、古典的確率（混合状態）と量子確率（純粋状態）を分ける境界はどこか？」**
という、より鋭い問いに答えることを目指す。

### 1.3 先行研究との位置づけ

| 研究 | 可逆性 | 構造 | 本研究の位置 |
|------|--------|------|--------------|
| Bennett (1973) | 可逆計算の基礎 | 古典 | 基盤として活用 |
| Toffoli (1980) | 可逆論理ゲート | 実数・古典 | 古典的限界を確認 |
| Deutsch (1985) | 量子チューリングマシン | 複素数を仮定 | 導出を目指す |
| Fredkin gate | 可逆・古典 | 実数・古典 | ビリヤードボール計算 |
| **本研究** | 可逆計算の限界 | 境界条件を探索 | 古典↔量子の分岐点 |

## 3. 研究戦略（レビュー反映版）

### 3.1 Phase 4: 可逆計算の代数構造（修正）

**目標**: 可逆計算の代数がシンプレクティック群に留まるか、複素構造に拡大するかを検証

#### 3.1.1 解析対象の変更：履歴付きSK → 可逆論理ゲート

**レビュー指摘**: 履歴付きSK (HSK) は状態空間が爆発し、合流が困難

**対策**: 有限状態で完結する**可逆論理ゲート（Toffoli, Fredkin）の合成**を主な解析対象とする

```
Toffoli: (a, b, c) → (a, b, c ⊕ (a ∧ b))
Fredkin: (a, b, c) → (a, (¬a ∧ b) ∨ (a ∧ c), (¬a ∧ c) ∨ (a ∧ b))
```

これらは：
- 有限状態（n-bit → n-bit）
- 行列表現が明確（2ⁿ × 2ⁿ 置換行列）
- 合流条件が定義しやすい

#### 3.1.2 シンプレクティック構造の検証

**新規タスク**: 生成される群が**シンプレクティック群 Sp(2n, ℝ)** の部分群かを確認

| 検証項目 | 方法 | 古典的な場合 | 量子的な場合 |
|----------|------|--------------|--------------|
| 群構造 | 生成元の閉包 | Sp(2n, ℝ) | U(n) |
| 保存量 | 不変量の特定 | シンプレクティック形式 ω | エルミート内積 |
| J² = -I | 複素構造探索 | 存在しない | 存在する |

#### 3.1.3 検証項目（修正版）

| 項目 | 方法 | 期待される結果 |
|------|------|----------------|
| **群の閉包** | Toffoli/Fredkin の生成 | 置換群 Sₙ に閉じる？ |
| **シンプレクティック埋め込み** | Sp(2n, ℝ) への埋め込み | 古典ハミルトン系と同型？ |
| **J² = -I の探索** | 系統的探索 | 複素構造の有無 |
| **保存される構造** | 不変量解析 | ω（シンプレクティック）or ⟨·,·⟩（エルミート）|

### 3.2 Phase 5: ハミルトニアンと干渉（大幅修正）

**目標**: 計算グラフのスペクトル解析により、干渉の有無を検証

#### 3.2.1 Landauer原理の限界（レビュー反映）

**レビュー指摘**: 抽象計算モデルには「温度」「エネルギー」が未定義。
Landauer原理から位相を導くには、物理的ハミルトニアンが必要。

**対策**: 計算グラフの**隣接行列**をハミルトニアンとして定義し、
スペクトル解析と量子ウォーク的アプローチを採用する。

#### 3.2.2 計算グラフのハミルトニアン

計算グラフ G = (V, E) に対して：

```
ハミルトニアン H = A (隣接行列)
時間発展 U(t) = exp(-iHt) = exp(-iAt)
```

これにより：
- **連続時間量子ウォーク**と等価な構造を自然に定義
- 計算構造から直接ハミルトニアンが導出される
- 干渉パターンの有無を検証可能

#### 3.2.3 検証方法（修正版）

| ステップ | 内容 | 成果物 |
|----------|------|--------|
| 1 | 計算グラフの隣接行列 A を構成 | 行列表現 |
| 2 | スペクトル解析（固有値・固有ベクトル） | スペクトル構造 |
| 3 | U(t) = exp(-iAt) による時間発展 | 干渉パターン |
| 4 | 古典ランダムウォークとの比較 | 量子性の判定 |

#### 3.2.4 Berry位相との接続

**追加検証**: 断熱的なパラメータ変化に相当する構造があるか探る

- パス空間のパラメータ化
- 断熱接続の定義
- Berry位相の計算

これはLandauer原理より位相に近い議論を可能にする。

### 3.3 Phase 6: 比較対象の絞り込み（修正）

**目標**: 3つの計算モデルの構造的比較

#### 3.3.1 比較対象の変更

**レビュー指摘**: Lambda計算とSKは本質的に同型（等価）のため、比較しても新知見が得にくい。

**修正**: 以下の3モデルに絞り込む

| モデル | 可逆性 | 決定論 | 予想される構造 |
|--------|--------|--------|----------------|
| **SK計算** | 非可逆 | 決定論的 | 古典（実数） |
| **可逆セルオートマトン (RCA)** | 可逆 | 決定論的 | 古典（シンプレクティック？） |
| **量子回路** | 可逆（ユニタリ） | 確率的 | 量子（複素） |

#### 3.3.2 可逆セルオートマトン (RCA) の採用理由

- **局所的な規則だけで可逆性を保つ**：SKの精神（局所書き換え）に近い
- **解析しやすい**：格子上の離散力学系として確立された理論がある
- **物理との接点**：格子ガス模型、離散時空模型との関連

#### 3.3.3 比較分析の焦点

| 比較軸 | SK | RCA | 量子回路 |
|--------|----|----|----------|
| 情報保存 | ✗ | ✓ | ✓ |
| 干渉 | ✗ | ? | ✓ |
| 代数構造 | 冪等的 | 置換群？ | U(n) |
| スペクトル | 実数 | 実数？ | 複素 |

**核心的問い**: RCAは古典（SK側）か量子（量子回路側）か、その中間か？

### 3.4 Phase 7: 非可換性と量子化（新規追加）

**目標**: 「計算の量子化」に必要な非可換構造を特定

#### 3.4.1 正準量子化との類比

物理学では、古典系（可逆）を量子系にするために**正準量子化**を行う：

```
古典: {q, p} = 1  (ポアソン括弧)
量子: [q̂, p̂] = iℏ  (交換子)
```

**核心的問い**: 計算モデルにおいて、この**「非可換性の導入」**に相当する操作は何か？

#### 3.4.2 候補となる非可換構造

| 候補 | 説明 | 検証方法 |
|------|------|----------|
| **書き換え順序の非可換性** | 異なる redex の適用順序 | 交換子 [Ŝ, K̂] の解析 |
| **パス合成の非可換性** | γ₁ ∘ γ₂ ≠ γ₂ ∘ γ₁ | パス代数の構造 |
| **観測の非可換性** | 異なる「観測」操作の順序依存 | 測定理論との対応 |

#### 3.4.3 量子コンビネータの不可能性

前研究で得られた洞察を深化：

```
量子K: K_q |ψ⟩|φ⟩ = |ψ⟩  ← 非ユニタリ（情報消去）→ 定義不可能
```

**重要な結論**: 量子計算には K のような情報消去がない。
これは単なる制約ではなく、**量子性の本質的特徴**である可能性。

#### 3.4.4 検証項目

| 項目 | 方法 | 成功基準 |
|------|------|----------|
| 交換子 [Ŝ, K̂] | 代数計算 | ≠ 0 かつ構造的意味 |
| パス代数の非可換性 | 圏論的解析 | 非自明なブレイド構造 |
| 「計算の量子化」の定式化 | 理論構築 | 明確な公理系 |

## 12. Phase 4 実験結果と計画への影響（2024年12月実施）

### 12.1 Phase 4 実験結果サマリー

Phase 4（可逆計算の代数構造）を実施し、以下の結果を得た：

| 検証項目 | 結果 | 判定 |
|----------|------|------|
| **Sp(2n,ℝ) 埋め込み** | 全要素が埋め込み可能 | 古典的 |
| **J² = -I（非自明解）** | 存在しない | 古典的 |
| **固有値** | 全て実数（1の冪根） | 古典的 |
| **連続的位相** | 存在しない | 古典的 |

#### 主要な発見

1. **可逆論理ゲート（Toffoli/Fredkin）は置換行列で表現される**
   - 全て直交行列（P^T P = I）
   - 全て自己逆（P² = I）

2. **シンプレクティック群 Sp(2n, ℝ) に埋め込み可能**
   - 埋め込み: P → [[P, 0], [0, P]]
   - 古典ハミルトン力学の対称群と同型

3. **複素構造 J² = -I は生成されない**
   - 置換行列の固有値は1の冪根（実数または複素共役ペア）
   - ±i は現れない → 非自明な J は存在しない
   - 複素係数を仮定すれば J = iI は存在するが、これは自明解

4. **離散性の限界**
   - 置換群は離散的
   - 連続的位相 e^{iθ} が定義できない
   - 干渉項 2Re(ψ₁*ψ₂) が計算できない
