# Phase 17: Experimental Implications

## 概要

Phase 13-16の理論的結果から導出される実験的予測を整理する。

## 予測1: 離散系における量子効果の境界

### 理論的背景
No-Go定理：N状態の離散系で、可逆的ダイナミクスは置換行列に限定され、Sp(2N,R)に埋め込まれる。

### 予測
```
離散系（N状態）で量子干渉が観測される場合、
状態空間はN-simplexを超えて拡張されている（A1が成立）。
```

### 検証方法
- **古典光**: 離散時間ウォーク → 干渉なし予測
- **量子光**: 離散時間量子ウォーク → 干渉あり観測
- **比較**: 同じ離散構造で、光子 vs 古典パルスの振る舞い

### 既存実験との関連
- Schreiber et al. (2012): 光量子ウォークでの干渉観測
- これは「A1が成立している」ことの間接的証拠

## 予測2: スピノル構造の必要性

### 理論的背景
Phase 13.3: Lorentz対称性 + スピン → SL(2,C)スピノル → A1

### 予測
```
スピン1/2粒子（電子、クォーク）の存在自体が、
状態空間がC²（複素2次元）であることの証拠。
```

### 検証方法
- **Stern-Gerlach実験**: スピン1/2の2値性
- **EPR/Bell実験**: エンタングルメント + Bell不等式の破れ
- **2π回転**: 中性子干渉計での-1位相

### 既存実験との関連
- Werner et al. (1975): 中性子の4π周期性を確認
- これは「スピノルが実在する」ことの直接的証拠

## 予測3: Contextualityの検証

### 理論的背景
Phase 14: KS Contextuality → A1

### 予測
```
Kochen-Specker型Contextualityが観測される系は、
必然的にA1を持つ（状態空間がsimplexを超える）。
```

### 検証方法
- **3量子ビット系**: Peres-Mermin不等式の破れ
- **連続変数系**: Contextuality witness

### 既存実験との関連
- Kirchmair et al. (2009): 単一イオンでのContextuality観測

## 予測4: 因果構造と量子性

### 理論的背景
Phase 15: 因果構造のみ → 古典的

### 予測
```
純粋に因果的（古典的）な系は、量子干渉を示さない。
因果構造を保ちながら量子干渉を得るには、A1の導入が必要。
```

### 検証可能性
- **原理的予測**: 古典コンピュータで真の量子干渉は再現不可
- **実験的検証**: 困難（「古典的」の定義による）

## 実験的予測の階層

| 予測 | 検証難易度 | 既存実験 | 新規性 |
|------|-----------|---------|--------|
| 1. 離散量子ウォーク | 中 | あり | 理論的枠組みの提供 |
| 2. スピノル構造 | 低（確立済み） | 多数 | 新解釈 |
| 3. Contextuality | 中 | あり | A1との関連付け |
| 4. 因果構造 | 高 | 限定的 | 概念的 |

## 結論

Phase 13-16の理論的結果は、既存の実験結果と整合的である：

1. **量子干渉の観測** → A1の成立を示唆
2. **スピン1/2粒子の存在** → スピノル構造（C²）の証拠
3. **Bell不等式の破れ** → 古典的（simplex）状態空間では不可能
4. **Contextuality** → A1の必然性の別証拠

新しい実験的予測として：
- **離散系での量子古典境界**: N状態系で、状態空間がN-simplexを超えるかどうかの直接的テスト
