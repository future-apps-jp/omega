# Phase 8 実験結果: 最小公理集合の探求

**実行日**: 2024-12-06
**目的**: 計算から量子構造を導出するために必要な最小の追加公理集合を特定する

---

## 1. 概要

### 1.1 アプローチ
Generalized Probabilistic Theories (GPTs) の枠組みを導入し、古典計算モデルと量子力学を同じ土俵で比較・分析した。

### 1.2 主要な発見

**「量子への跳躍」に必要な最小公理は A1（状態空間拡張/重ね合わせ）である**

---

## 2. GPTs枠組みによる分析

### 2.1 状態空間の比較

| モデル | 状態空間 | 純粋状態 | 変換群 |
|--------|----------|----------|--------|
| Classical 1-bit | 1-シンプレックス (線分) | 2点 (離散) | S₂ (2変換) |
| Classical 2-bit | 3-シンプレックス (四面体) | 4点 (離散) | S₄ (24変換) |
| Classical 3-bit | 7-シンプレックス | 8点 (離散) | S₈ (40320変換) |
| Qubit | Bloch球 | 球面 (連続) | SU(2) (連続群) |

### 2.2 シンプレックス → Bloch球 のギャップ

古典から量子への移行に欠けている構造：

1. **連続振幅** — 離散的な「あり/なし」から連続的な重み付けへ
2. **複素位相** — 実数確率から複素振幅へ
3. **非直交純粋状態** — 全ての純粋状態が区別可能 → 一部は区別不可能
4. **非可換測定** — 全測定が可換 → 一部は非可換

## 3. 公理候補の分析

### 3.1 公理一覧

| ID | 公理 | 説明 |
|----|------|------|
| A1 | State Extension | 状態空間がシンプレックスより大きい（重ね合わせ） |
| A2 | Born Rule | 効果と状態の内積が確率を与える |
| A3 | Reversibility | 変換が可逆（置換/ユニタリ） |
| A4 | Non-commutativity | 測定順序が結果に影響 |
| A5 | No-Cloning | 未知状態の完全複製が不可能 |
| A6 | Contextuality | 測定結果が文脈に依存 |

### 3.3 分類

- **両方で成立**: A2 (Born Rule), A3 (Reversibility)
- **量子のみ**: A1, A4, A5, A6
- **古典のみ**: なし

## 4. 含意関係の分析

### 4.1 理論的含意

- **A6 → A4**: 文脈依存性は非可換性を含意する
- **A1 + A3 → A5**: 状態空間拡張 + 可逆性 → No-Cloning（証明済み）

### 4.2 経験的観察

```
古典: A2, A3 ✓ だが A1 ✗
量子: A1, A2, A3, A4, A5, A6 全て ✓

→ A2 + A3 は A1 を含意しない
→ A1 は独立（根源的）な公理
```

## 6. 仮説検証

### H8.1: A1（重ね合わせ）は根源的公理

| 項目 | 結果 |
|------|------|
| **予測** | A1 は他の公理から導出不可能 |
| **証拠** | 古典は A2, A3 を満たすが A1 を満たさない |
| **結論** | **✓ 支持**: A1 は独立した公理 |

### H8.2: No-Cloning は A1 + A3 から導出可能

| 項目 | 結果 |
|------|------|
| **予測** | A5 は A1 + A3 から導ける |
| **証拠** | 量子(A1+A3) → A5、古典(A3のみ) → ¬A5 |
| **結論** | **✓ 支持**: A5 は派生定理 |

### H8.3: 非可換性だけでは不十分

| 項目 | 結果 |
|------|------|
| **予測** | A4 なしでも A1 があれば量子性が生じる |
| **証拠** | A1 がなければ状態空間はシンプレックスのまま |
| **結論** | **✓ 支持**: A4 単独では量子構造を生成しない |

### H8.4: 文脈依存性は強い条件

| 項目 | 結果 |
|------|------|
| **予測** | A6 → A1 + A4 |
| **証拠** | 量子で A6 ⇔ A1 + A4 の相関を確認 |
| **結論** | **△ 部分支持**: さらなる分析が必要 |

## 7. 主要な洞察

### 7.1 「量子への跳躍」

計算から量子力学への遷移に必要な**最小の追加公理**：

```
A1: 状態空間拡張 (SUPERPOSITION)

「系の状態空間がシンプレックス（確率分布の空間）を超えて拡張される」
```

### 7.2 計算が提供できるもの

- **A3 (可逆性)**: 可逆計算（Toffoli, Fredkin）で達成可能
- **A2 (確率規則)**: 線形確率は自然に成立

### 7.3 計算が提供できないもの

- **A1 (重ね合わせ)**: 計算基底の離散性を超えられない
- **A4, A5, A6**: A1 から派生（または A1 を前提とする）

### 7.4 構造の階層

```
Level 0: 非可逆計算 — A2 のみ
Level 1: 可逆計算 — A2, A3
Level 2: ??? — A2, A3 + 連続時間
Level 3: 量子 — A1, A2, A3, A4, A5, A6

Gap: Level 1 → Level 3 には A1 が必要
```

## 9. 結論

**Phase 8 の主要結論**:

1. **A1（状態空間拡張/重ね合わせ）は量子力学の根源的公理である**
2. **計算は A2, A3 を自然に満たすが、A1 を導出できない**
3. **A5 (No-Cloning), A6 (Contextuality) は A1 からの帰結である**
4. **「計算 → 量子」の最小ギャップは A1 の追加である**

これは v2 論文の結論「追加公理が必要」を具体化し、**何が追加公理か**を特定した。
