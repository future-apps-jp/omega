# Phase 1B 実験結果: パス空間のホロノミー解析

**実施日**: Phase 1B  
**目的**: SK計算のパス空間に「接続」を定義し、そのホロノミーから位相構造（U(1)）が現れるかどうかを検証

---

## 実験概要

### 理論的背景

- **ホロノミー**: パラメータ空間のループに沿った平行移動の効果
- **Berry位相**: 量子力学における幾何学的位相
- **U(1) 群**: 量子力学の位相を記述する群

### 検証方法

1. SK式から Multiway Graph を構築
2. 同じ始点・終点を持つパスペアから「ループ」を抽出
3. 各ループに対してホロノミー（位相差）を計算
4. 複数の接続（位相割り当て方法）で比較

### 使用した接続（位相割り当て方法）

| 接続名 | 定義 |
|--------|------|
| `constant_S_K` | S → exp(iπ/4), K → exp(-iπ/4) |
| `constant_S_only` | S → exp(iπ/2), K → 1 |
| `depth_dependent` | 位相が式内の深さに依存 |
| `complexity` | 位相が式のサイズ変化に依存 |
| `info_erasure` | K → exp(iπ), S → 1（情報消去に π 位相）|

---

## 実験結果

### 各式の結果サマリー

| Expression | ループ数 | 非自明ホロノミー | U(1) 候補 |
|------------|----------|------------------|-----------|
| `S (K a) (K b) c` | 1 | いいえ | いいえ |
| `(K a b) (K c d)` | 1 | いいえ | いいえ |
| `S (K a b) c d` | 1 | はい (depth) | いいえ |
| `(K a b) (K c d) (K e f)` | 15 | いいえ | いいえ |
| `S S K a b c` | 0 | N/A | N/A |
| `S (K a) (K b) (S c d e)` | **120** | **はい** | **はい** |

### 🔔 重要な発見: `S (K a) (K b) (S c d e)`

この式で最も複雑なパス空間が生成され、**U(1) 構造の候補**が見つかりました。

#### 接続ごとの結果

| 接続 | 非自明 | U(1) 候補 | 位相範囲 | 異なる位相数 |
|------|--------|-----------|----------|--------------|
| `constant_S_K` | ✓ | ✓ | [-π/2, π/2] | 5 |
| `constant_S_only` | ✓ | ✓ | [-π, π] | 5 |
| `depth_dependent` | ✓ | ✓ | [-π/3, π/3] | 8 |
| `complexity` | ✗ | ✗ | [0, 0] | 1 |
| `info_erasure` | ✗ | ✗ | [0, 0] | 1 |

#### ホロノミー位相の例

```
constant_S_K:
  -π/4  (0.25π)
  -π/2  (0.50π)
  +π/4  (0.25π)
  ...

constant_S_only:
  -π/2  (0.50π)
  -π    (1.00π)
  +π/2  (0.50π)
  ...
```

## 結果の解釈

### ⚠️ 重要な警告

**観測された U(1) 構造は「導出」されたものではない。**

理由：
1. **位相は接続で仮定されている**: S と K に位相を「割り当てた」
2. **接続依存性**: `complexity` と `info_erasure` では U(1) が現れない
3. **任意性**: どの接続が「正しい」かの基準がない

### 何が確認されたか

1. **パス空間の構造は複雑**: 複雑な式では多数のループが存在
2. **非自明なホロノミーは可能**: 適切な接続を選べば非自明な位相差が生じる
3. **U(1) 的構造は接続選択に依存**: 「自然な」接続の定義が必要

### 何が確認されなかったか

1. **位相の「導出」**: 位相は計算から自然に出てきたのではなく、仮定された
2. **接続の一意性**: 複数の接続が考えられ、どれが「正しい」か不明
3. **物理との対応**: 得られた位相が物理的干渉と対応するかは未検証

---

## 考察: なぜこの結果が重要か

### 肯定的側面

1. **パス空間は十分に豊か**: ループが存在し、ホロノミーが計算可能
2. **U(1) との整合性**: 適切な接続を選べば U(1) 構造が現れる
3. **接続の多様性**: 異なる物理的解釈を試す余地がある

### 否定的側面

1. **自己完結的な導出ではない**: 位相を「外から」導入している
2. **Phase 1A と同じ問題**: 複素数を最初から仮定している
3. **「なぜこの接続か」の答えがない**: 接続の選択が恣意的

---

## 結論

### Phase 1B の結論

> **パス空間のホロノミーからは、位相構造を「導出」できなかった。**
> 
> ただし、以下は確認された：
> 1. SK計算のパス空間は十分に複雑
> 2. 適切な接続を仮定すれば U(1) 構造が現れる
> 3. 接続の選択が「物理的に正しい」かは未解決

### Phase 1A との比較

| アプローチ | 結果 | 問題点 |
|------------|------|--------|
| Phase 1A（代数的） | J² = -I の候補は自明解 | 複素係数を最初から使用 |
| Phase 1B（幾何学的） | U(1) は接続に依存 | 位相を最初から仮定 |

両アプローチとも、複素数/位相を**導出**するのではなく**仮定**している。

### ホロノミーの計算式

```
ホロノミー(ループ) = exp(i∑θ_path1) / exp(i∑θ_path2)
                  = exp(i(∑θ_path1 - ∑θ_path2))
```

ここで θ は各操作に割り当てられた位相。
